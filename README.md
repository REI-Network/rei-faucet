# gxchain2-fauct
 The gxchain2's fauct

### How to get token?
Use the `http://localhost:3000/send?address=youraddress`
### Error code
- `0` Sucessful
- `1` Api call limit in 24 hours
- `2` Ask token limit in 24 hours
- `3` Invalid address
- `4` Transfer failed
- `5` Other errors
- `6` Balance not enough
